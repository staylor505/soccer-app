# Soccer Management App

![Kicking Soccer Ball](frontend/public/soccer.jpeg)

## First MERN App Development

### Highlights:

- Create backend NodeJS and Express Server
- Handle routes for players
- Create frontend with ReactJS
- Use Axios to handle form data
- Use MongoDB to store player info

## Getting Started

### Prerequisites

- Node.js
- Express.js
- MongoDB
- React.js
- npm or yarn

### Installation

1. Clone the repository:

   ```sh
   git clone <repository-url>
   ```

2. Install backend dependencies:

   ```sh
   cd backend
   npm install
   ```

3. Install frontend dependencies:
   ```sh
   cd ../frontend
   npm install
   ```

### Running the Application

1. Start the backend server:

   ```sh
   cd backend
   npm start
   ```

2. Start the frontend development server:
   ```sh
   cd ../frontend
   npm start
   ```

### API Endpoints

- `GET /players` - Retrieve all players
- `POST /players` - Add a new player
- `PUT /players/:id` - Update a player
- `DELETE /players/:id` - Delete a player

- Node.js

- React
